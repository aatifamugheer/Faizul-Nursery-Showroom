# 🌿 Faizul Nursery Showroom

A full-stack **Plant Nursery E-Commerce Website** built using **PHP, MySQL, HTML, CSS, Bootstrap, JavaScript, and AJAX**. The platform allows users to browse plants, add products to the cart, place orders, and make secure online payments through an intuitive and responsive interface.

## ✨ Features

- 🌱 Browse plant categories and products
- 🔍 Search and filter plants
- 👤 User Registration & Login
- 🛒 Shopping Cart
- 💳 Secure Checkout & Online Payment
- 📦 Order Management
- ❤️ Wishlist
- 📱 Fully Responsive Design
- 🔐 Admin Dashboard for Product & Order Management

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript, AJAX
- **Backend:** PHP
- **Database:** MySQL
- **Payment Gateway:** Razorpay

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/aatifamugheer/Faizul-Nursery-Showroom.git
```

2. Navigate to the project directory:

```bash
cd Faizul-Nursery-Showroom
```

3. Import the MySQL database.

4. Configure the database connection in the PHP configuration file.

5. Start Apache and MySQL using XAMPP/WAMP.

6. Open the project in your browser:

```text
http://localhost/Faizul-Nursery-Showroom/
```
